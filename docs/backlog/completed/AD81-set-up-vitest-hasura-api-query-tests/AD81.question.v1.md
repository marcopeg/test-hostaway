### 1. Should `make test.api` assume Hasura is already running and initialized, or should it run any backend lifecycle commands first?

but in between test you should reset the state using `make migrate.rebuild && make seed`.
You can either look at what these commands do and use hasura's apis, or simply run the make commands in a child process.
