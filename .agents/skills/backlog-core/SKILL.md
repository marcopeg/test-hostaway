# Backlog Core

`backlog-core` is shared infrastructure for the backlog skills.

Use it to host reusable scripts and helper code that should not belong to a single operational skill such as `backlog-migrate`, `backlog-plan`, or `backlog-execute`.

Its reusable scripts live under `scripts/`.
